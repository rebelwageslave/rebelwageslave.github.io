Create a basic database and table structure for forum

Before creating a database connection you need to login to the mysql server. You can do this via phpmyadmin a GUI tool to access your mysql instance. First of all we need to create a database named forum

Step 1: Open phpmyadmin, click on the left side, and click on the icon.

Step 2: Set the database name as forum.

Step 3: Now we have a database ready, we need to create a table in the mysql, to create a table you need to execute a mysql query, you can execute the mysql query by clicking on this option.

The query you need to paste in this box is the below.

CREATE TABLE `forum`.`question` ( `id` INT(100) NOT NULL AUTO_INCREMENT , `question` VARCHAR(255) NOT NULL , `answer` TEXT NOT NULL , PRIMARY KEY (`id`));

Click on the go button, if everything goes well we should have a table like this.

PHP, MYSQL are two different process at the operating system level, to communicate with one another PHP has a predefined function which can be used to connect to MYSQL database.

The function is 

mysqli_connect($host, $username, $password, $database ...some other arguments)

so we need to provide $host, $username, $password, $database values in order to use this function

Note: If you didnt setup password for mysql you can pass null for $password variable

Lets try to establish a database connection, type the below code in index.php.

<?php

$host = 'localhost';

$username = 'phpmyadmin';

$password = 'some_pass';

$database = 'forum';

$database_connection = mysqli_connect($host, $username, $password, $database);


if ( $database_connection === false ) {
	echo "Database connection failed";
}
else {
	echo "Database Connection success";
}


?>

the mysqli_connect according to documentation will return a object when the database connection is created, if its not then it will return false.

Set $username and $password value for the code and check if you see the below message in the screen

Database Connection success

if you see this message then we are good to proceed in to next step, if not then please check your connection variables or comment down your problem you see.

Now we need to add code to save the data from this input in the database.

Whenever you submit the form by clicking on Create question button, the following process happens

The browser makes a request to the index.php which is present on the server with post data which is later parsed by php in to variable named $_POST[‘question’] in the index.php, we need to save this variable to database.

Note 1: The browser sends the variable with name 'question' because the form field is defined with this name <input ... name="question"/>
Note 2: we should also note that this variable doesnt present on every page load on index.php, so we need to always check if the variable is present or not.

lets add some more code to get the variable and print it out on submit.

<title>Welcome to my forum | Add your question</title>
<form method="POST">
	<input type="text" placeholder="Enter your question here" name="question">
	<input type="submit" value="Create question">
</form>

<?php

$host = 'localhost';

$username = 'phpmyadmin';

$password = 'some_pass';

$database = 'forum';

$database_connection = mysqli_connect($host, $username, $password, $database);

// isset and ! empty check to see if variable is not empty and is set.

if ( isset($_POST['question'] ) && !empty($_POST['question'])) {

	$question = $_POST['question'];

	echo $question; // This command prints out the variable

}

If you type the question on the input and click on create question, you should be able to see the question.

Congrats, now all we have to do this save the question to the database, to do that we need to create a mysql query and send the query to the mysql server.

// isset and ! empty check to see if variable is not empty and is set.

if ( isset($_POST['question'] ) && !empty($_POST['question'])) {

	$question = $_POST['question'];

	// This query will add the question to the mysql database, note that we are substituting
	// the $query variable in the below query.
	$query = "INSERT INTO `question` (`id`, `question`, `answer`) VALUES (NULL, '$question', '') ";

	mysqli_query($database_connection, $query);

}

Now go http://localhost/forum/, Add a question, click on submit.

After you submit, Go to phpmyadmin and check the question table.

Now we have successfully completed the step of saving question to database.

Show all saved questions in the index.php

Its great that we now have ability to save the question, but we will also need the ability to see all saved question and answer, to do that we need to add additional code in index.php.

$questions_query = "SELECT * FROM `question`";
$questions_result = mysqli_query($database_connection, $questions_query);

while ( $question_data = mysqli_fetch_assoc($questions_result) ) {
	$question = $question_data['question'];
	$answer = $question_data['answer'];

	echo "<div class='single-question'>
		<b>$question</b>
		<p>$answer</p>
	</div>";
}

we would now be able to see the saved questions

We have now successfully completed the step 3, the only remaining step is allowing the user to add the answer for a question.

Allow user to add answer for a question.

we are going to use the question.php which we created before to acheive this functionality.

To do that we need to redirect the user from index.php to question.php, we are going to use the <a> element to link a question from index.php to question.php and also add a url parameter id in the index.php, so we alter the code like this in index.php.

while ( $question_data = mysqli_fetch_assoc($questions_result) ) {
	$question = $question_data['question'];
	$answer = $question_data['answer'];
		$id = $question_data['id'];

	echo "<div class='single-question'>
		<b><a href='question.php?id=$id'>$question</a></b>
		<p>$answer</p>
	</div>";

}

Now we would see the question with question link with the id param, the url should look like this.

http://localhost/forum/question.php?id=4

The id parameter denotes the question id which is needed in question.php to show the question and answer, so lets add some code on question.php to show the question and a answer box.

The url param id is parsed by php and provided to us in the format $_GET[‘id’] , we are going to use the variable to show the question and answer from db. We need create a sql query to get the question and answer data for id.

To access the database we need to have our database connection code from index.php, lets copy it down to question.php

<?php

$host = 'localhost';

$username = 'phpmyadmin';

$password = 'some_pass';

$database = 'forum';

$database_connection = mysqli_connect($host, $username, $password, $database);


$id = $_GET['id']; // we get the id url parameter here.

we will write the query to get the data for id and print on the page 

$sql = "SELECT * FROM `question` WHERE id='$id'";

$question_query = mysqli_query($database_connection, $sql);

$question_data = mysqli_fetch_assoc($question_query);

$question = $question_data['question'];
$answer = $question_data['answer'];

echo "
<form method='POST'>
<div class='single-question'>
		<b>$question</b>
		<br/><br/>
		<textarea name='answer' cols='100' rows='4'>$answer</textarea>
		<br/><br/>
	</div>
	<input type='submit'  value='Update answer'/>
	</form>
";

this should render a form on the page

This form alone doesnt function, we need to add code in php side to save the data on question table, we are going to use the update query.

if you noted this

<textarea name='answer' ...>

then we would know that we need to get the $_POST[‘answer’] variable to update the answer for the question. lets add the code to save the data just like we did for question

// isset and ! empty check to see if variable is not empty and is set.

if ( isset($_POST['answer'] ) && !empty($_POST['answer'])) {

	$answer = $_POST['answer'];

	// This query will add the question to the mysql database, note that we are substituting
	// the $query variable in the below query.
	$query = "UPDATE `question` SET answer='$answer' WHERE id='$id'";

	mysqli_query($database_connection, $query);

	echo "Answer successfully updated";


}
