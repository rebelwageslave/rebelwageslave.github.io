PHP ஐப் பயன்படுத்தி முதன்மை எண்ணைக் கண்டறிதல்


பிரதான எண் என்றால் என்ன?

ஒரு பிரதான எண் (அல்லது ஒரு பிரதான) என்பது 1 ஐ விட அதிகமான இயற்கை எண், இது இரண்டு சிறிய இயற்கை எண்களின் தயாரிப்பு அல்ல. முதன்மையானதல்ல 1 ஐ விட அதிகமான இயற்கை எண் ஒரு கூட்டு எண் என்று அழைக்கப்படுகிறது. எடுத்துக்காட்டாக, 5 முதன்மையானது, ஏனெனில் 1 × 5 அல்லது 5 × 1 என ஒரு தயாரிப்பாக எழுதுவதற்கான ஒரே வழிகள் 5 ஐ உள்ளடக்கியது. இருப்பினும், 4 கலப்பு ஆகும், ஏனெனில் இது ஒரு தயாரிப்பு (2 × 2), இதில் இரு எண்களும் 4 ஐ விட சிறியவை. எண்கணிதத்தின் அடிப்படைக் கோட்பாட்டின் காரணமாக முதன்மைக் கோட்பாடு எண் கோட்பாட்டில் மையமாக உள்ளது: 1 ஐ விட அதிகமான ஒவ்வொரு இயற்கை எண்ணும் ஒரு பிரதம அல்லது அவற்றின் வரிசைக்கு தனித்துவமான பிரைம்களின் தயாரிப்பாக காரணியாக்கப்படலாம்.

PHP என்றால் என்ன?

PHP என்பது ஒரு பொது நோக்கம் ஸ்கிரிப்டிங் மொழி குறிப்பாக வலை அபிவிருத்திக்கு ஏற்றது.

Code:

<?php
$num = 1 ;
while ($num < 100 ) {
    $count=0;
    for ( $i=1;$i<=$num;$i++) {
        if (($num%$i)==0) {
            $count++;
        }
    }
    if ($count<3) {
    echo $num." , ";
    } $num=$num+1;
}
?>
